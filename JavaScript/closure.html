Closure

A closure is when a function remembers variables from its outer scope, even after that outer function has finished executing. 
        Example
        
            function outer() {
            count = 0;

            function inner () {
                count++;
                console.log(count);
            }
            return inner;
        }
        let a = outer()
        a();
        a();
        a();

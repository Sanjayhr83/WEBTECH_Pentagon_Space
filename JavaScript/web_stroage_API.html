Web Storage API :

Eachh and every browser provide in-built storage to store data up to a certain limit(10 MB).

There are 2 types of web storage :

  • localStorage (Permanent storage)
  • sessionStorage (Temporary storage)

1. localStorage

    //To store data 

    syntax : 

        localStorage.setItem("key",value); 


    //To get data 

    syntax : 

        localStorage.getItem("key");

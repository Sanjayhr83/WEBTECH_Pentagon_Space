BigInt

BigInt is a built-in object in JavaScript that provides a way to represent whole numbers larger than 2^53 - 1, which is the largest number that can be safely represented with the Number type. BigInt can be created by appending 'n' to the end of an integer literal or by using the BigInt() constructor.

Example of creating a BigInt:

const bigInt1 = 123456789012345678901234567890n; // Using 'n' suffix
const bigInt2 = BigInt("123456789012345678901234567890"); // Using BigInt constructor

BigInt supports various arithmetic operations such as addition, subtraction, multiplication, and division. However, it does not support floating-point numbers or the Math object methods.

Promises in JS

A Promise is an object that represents the future result of an asynchronous operation.

Real-world Analogy: imagine ordering online food :
Order placed → Pending
Food delivered → Fulfilled
Order cancelled → Rejected

🔄 Promise States
A Promise has 3 states: 

        | State     | Meaning |
        | --------- | ----------------------- |
        | Pending   | Operation still running |
        | Fulfilled | Operation successful    |
        | Rejected  | Operation failed        |

🧱 Creating a Promise :
Syntax :
let promise = new Promise((resolve, reject) => { });

simple example :
let promise = new Promise((resolve, reject) => {
let success = true;
if(success) {
resolve("Task completed");
} else {
reject("Task failed");
}
});

🔹 resolve() and reject() 

            | Function    | Purpose        |
            | ----------- | -------------- |
            | `resolve()` | Success result |
            | `reject()`  | Failure result |

🔹 Handling Promises

✅ .then() → Success
promise.then(result => {
console.log(result);
});

❌ .catch() → Error
promise.catch(error => {
console.log(error);
});

🔁 .finally()
promise.finally(() => {
console.log("Finished");
});

Full example : 

        let promise = new Promise((resolve, reject) => {

            setTimeout(() => {
                resolve("Data received");
            }, 2000);

        });

        promise
        .then(data => console.log(data))
        .catch(err => console.log(err))
        .finally(() => console.log("Done"));
🔍 Output After 2 Seconds
Data received
Done


💡 Easy Memory Trick : 

            Promise = Future value
            then() = success
            catch() = failure
            finally() = always run

Real example(fake API) : 

            function fetchData() {

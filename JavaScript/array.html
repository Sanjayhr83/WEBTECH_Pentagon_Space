Array and Array methods

Array :
Array is a collection of Homogeneous and Heterogeneous types of data

Hemogeneuos :
Hemogeneuos means array that contains same types of data

Heterogeneous :
Heterogeneous means array that contains different types of data

There is 2 ways to declare Array


        1.literal method
        2.Array constructor

1.literal method :- let users =["Rahul","Shashank","Shaarath","Rakshith","Neharu"];

2.Array constructor :- let user = new Array("Rahul","Shashank","Shaarath","Rakshith","Neharu");

Methods in Array :-

  1. push()
  2. pop()
  3. unshift()
  4. shift()
  5. splice()
        1.push() → Add at end
        
            let arr = [1, 2];
            arr.push(3);
            console.log(arr);
        
    
            2.pop() → Remove from end
            
                let arr = [1, 2, 3];
                arr.pop();
                console.log(arr);
            
        
            3.unshift() → Add at beginning
            
                let arr = [2, 3];
                arr.unshift(1);
                console.log(arr);
            
        
            4.shift() → Remove from beginning
            
                let arr = [1, 2, 3];
                arr.shift();
                console.log(arr);
            
        
            5.splice () ->
                syntax : array_name.splice(start_index,deletecount,item1,item2,.......);
                
                    let arr = [1, 2, 3, 4];
                    let result = arr.slice(1, 3);
                    console.log(result);

                    let players = ["Virat", "MSD", "Chahal", "Rahul"];
                    console.log(players);
                    players.push("Bumrah", "Siraj"); //Add at end
                    console.log(players);
                    players.unshift("Kapil dev");   //Add at beginning
                    console.log(players);
                    players.pop();
                    console.log(players);    //Remove from end
                    players.shift()
                    console.log(players);   //remove from beginning

                    //Removing elements using splice
                    let arr = ["Virat", "MSD", "Chahal", "Rahul"];
                    arr.splice(1, 2);   //delete "MSD","Chahal"
                    console.log(arr);   //['Virat', 'Rahul']

                    //Replacing the elements using splice
                    let arr1 = ["Virat", "MSD", "Chahal", "Rahul"];
                    arr1.splice(2,1,"Ashwin");
                    console.log(arr1);  //['Virat', 'MSD', 'Ashwin', 'Rahul']

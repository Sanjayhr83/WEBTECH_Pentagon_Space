Scope defines where a variable is accessible in your code.

Types of Scope in JavaScript

  1. Global scope
  2. Function/local scope
  3. block scope

1.Global Scope :

A variable declared outside all functions.

        
            let name = "Sanjay";

            function greet() {
            console.log(name);
            }

            greet();

Function/local scope :

Variables declared inside a function.

        
            function test() {
            let x = 10;
            console.log(x);
            }

            test();
            console.log(x); // ❌ Error
        
        👉 x is only available inside the function

Block Scope { } :

Variables declared with let or const inside {}.

        
            {
            let a = 5;
            console.log(a); // works
            }

            console.log(a); // ❌ Error
        
        ✔ Only accessible inside the block
        
            {
            var x = 10;
            }
            console.log(x); // ✅ works
👉 var is NOT block scoped

Summary


        | Scope  | Accessible Where |
        | ------ | ---------------- |
        | Global | Everywhere       |
        | Local  | Inside function  |
        | Block  | Inside `{}`      |

Function :

Functions are blocks of code designed to perform a particular task when called.

Types of Functions :

  1. Anonymous Functions
  2. Named Functions
  3. Functional Expressions
  4. First-Class Functions
  5. IIF(Immediately Invoked Function)
  6. Arrow Functions
  7. Nested Functions
  8. Higher-Order Functions
  9. Call-back Functions
  10. Generator Functions

Anonymous Functions:

Functions without a name, often used for short, one-time operations.We can't execute them without assigning them to a variable.
            
            // Anonymous function example
            we can not directly execute this because it does not have a name, but we can assign it to a variable and then execute it.
            function ()
            {
                console.log("This is an anonymous function.");
            }

Named Functions:

Functions that have a name and can be called by that name.
            
            // Named function example
            function namedFunction() {
                console.log("This is a named function.");
            }
            namedFunction(); // Output: This is a named function.

Functional Expressions:

Functions that are assigned to variables or passed as arguments to other functions.
            
            // Functional expression example
            let x = function exp ()
            {
                console.log("This is a functional expression.");
            }
            x(); // Output: This is a functional expression.

            let add = function(a, b) {
            return a + b;
            };
            console.log(add(2, 3)); // Output: 5

First-Class Functions:

Functions that can be treated as first-class citizens, meaning they can be assigned to variables, passed as arguments, and returned from other functions.
            
            // First-Class function example
            function firstClassFunction() {
                console.log("This is a first-class function.");
            }
            firstClassFunction(); // Output: This is a first-class function.

IIF(Immediately Invoked Function):

Functions that are defined and immediately invoked, often used to create a new scope.
            
            // IIF(Immediately Invoked Function) example
            (function ()
            {
                console.log("This is an IIF(immediate invoked function) function.");
            })
            (); // Output: This is an IIF(immediate invoked function) function.

Arrow Functions:

Functions that use the => syntax, providing a shorter way to write function expressions.
            
            // Arrow function example
            const arrowFunction = () => {
                console.log("This is an arrow function.");
            };
            arrowFunction(); // Output: This is an arrow function.

return Functions:

Functions that return a value when called.
            
            // return function example
           function y()
        {
            return "Hello world!";
        }
        console.log(y()); // Output: Hello world!
        let q= y();
        console.log(q); // Output: Hello world!
        console.log(y); // Output: [Function: y]

Higher-Order Functions:

Functions that take other functions as arguments or return functions as their result.
            
            // Higher-Order function example
            function higherOrderFunction(func) {
                return function() {
                    console.log("This is a higher-order function.");
                    func();// Call the callback function
                }
            }
            const callbackFunction = () => {
                console.log("This is a callback function.");
            }
            higherOrderFunction(callbackFunction)(); // Output: This is a higher-order function.
            // Output: This is a callback function.

Callback Functions:

Functions that are passed as arguments to other functions and are executed after a certain event or condition is met.
            
            // Callback function example
            function fetchData(callback) {
                setTimeout(() => {
                    const data = "Data fetched from server";
                    callback(data); // Call the callback function with the fetched data
                }, 2000);
            }
            function displayData(data) {
                console.log(data); // Output: Data fetched from server
            }
            fetchData(displayData);

Generator Function :

It is special type of function to generate sequence of values.
when ever use yeild kwyword to function that generator function.yield pause the execution the next keyword use to resume the execution where it is pause.
            
            function* generatorFunction() {
            yield "First value";
            yield "Second value";
            yield "Third value";
        }
        const generator = generatorFunction();
        console.log(generator.next().value); // Output: First value

DOM Selectors :

  • document.getElementById()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
  • document.querySelector()
  • document.querySelectorAll()

document.getElementById()

document.getElementById() method is used to target element by id and it return single element.

        
        let a = document.getElementById("byid");
        a.style.color = "blue";
        a.style.fontSize = "30px";
        a.style.fontFamily = "Arial, sans-serif";
        a.style.backgroundColor = "lightgray" ;
        let para = document.getElementById("para");
        para.innerText = "This is a paragraph with id 'para' and it is styled using JavaScript DOM selector getElementById() method.";

document.getElementsByClassName()

document.getElementsByClassName() is used to target elements by class name and it return HTML collections like an array.

        
        let b =document.getElementsByClassName("byclass");
        b[0].style.color = "green";
        b[0].style.backgroundColor ="lightgray";
        b[1].style.color = "red";

document.getElementsByTagName()

document.getElementsByTagName() is used to target elements by tag name and it return HTML collections.

        
        let c = document.getElementsByTagName("h2");
        c[0].style.color = "purple";
        let d = document.getElementsByTagName("h3");
        d[0].style.backgroundColor = "lightblue";
        d[0].style.fontStyle = "italic";

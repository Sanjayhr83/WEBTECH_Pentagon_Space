JSON :

JSON stands for: JavaScript Object Notation
It is a lightweight data format used to store and exchange data between:
Browser ↔ Server
Frontend ↔ Backend
APIs ↔ Applications 

            Simple Definition : JSON is a way to write data in key : value format.

Example:

{
  "name": "Sanjay",
  "age": 21,
  "city": "Bangalore"
}

Here:

"name" → key
"Sanjay" → value
Why JSON is Used :
Because it is:
✅ Simple
✅ Easy to read
✅ Easy to send over internet
✅ Supported by almost all programming languages
That’s why APIs mostly send data in JSON format.
JSON Rules
Rule 1: Data is in key-value pairs
{
"name": "John"
}
Rule 2: Keys must be inside double quotes
✅ Correct
{
"name": "John"
}
❌ Wrong
{
name: "John"
}
Rule 3: String values also use double quotes
{
"city": "Delhi"
}
Rule 4: Multiple values separated by comma
{
"name": "John",
"age": 25
}
JSON Data Types
Type Example
String "name": "Sanjay"
Number "age": 21
Boolean "isStudent": true
Array "skills": ["HTML", "CSS"]
Object "address": {"city": "Bangalore"}
Null "phone": null

Example of Full JSON
{
"name": "Sanjay",
"age": 21,
"isStudent": true,
"skills": ["HTML", "CSS", "JavaScript"],
"address": {
"city": "Bangalore",
"pincode": 560001
}
}

json.stringify()

json.stringify(): it is convert js objects into json data [JSON string]

json.parse()

json.parse(): it converts json string back to js objects

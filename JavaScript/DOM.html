DOM : Document Object Model

DOM is a programming interface provided by the browser that allows JavaScript to interact with the HTML and CSS of a webpage.

DOM converts HTML elements into Objects

It is interact with document.

features of DOM :-

  • To modify HTML element
  • To change Style
  • To add,delete or modify,edit the elements
  • To handle events

BOM : Browser Object Model

BOM is a JS to interact with browser(outside the pae content) like browser,window,location,history etc...

It is interact with browser

BOM objects :

  • Window(global object)
  • navigation(info about browser)
  • screen(info about screen resolution)
  • history(browser history)
  • location(info about urls)

DOM Tree :

whenever browser loads an HTML documents browser will create a tree structure this structure is called DOM Tree.

the html code convert into tree structure

        Document
        └── html
            ├── head
            │    ├── meta
            │    │    └── charset="UTF-8"
            │    └── title
            │         └── "My Page"
            │
            └── body
                ├── h1
                │    └── "Hello"
                │
                └── p

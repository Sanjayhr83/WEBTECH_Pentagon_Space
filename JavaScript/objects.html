Objects :

object is a real world entity used to store the data in the form of key:value pairs

There are two ways to create objects in JavaScript:
  1. Object Literal
  2. Object Constructor

1. Object Literal : An object literal is the simplest and most common way to define a JavaScript object.
syntax : let object_name = {key1 : value1, key2 : value2, key3 : value3, ...};

        Example :
        
            let product = {
                name : "Phone",
                color : "Purple",
                price : 21000,
                features : {
                    RAM : "8 + 8 GB",
                    ROM : "128 GB",
                    Charging : "80 W",
                    Cable type : "Type C"
                }
            }
            console.log(product);
            console.table(product);   // it will print the object in table format

2. object constructor : An object constructor used to create a object new keyword
syntax : let object_name = new Object();

        Example :
        
            let person = new Object( {
                name : "Ram",
                age : 23,
                area: "Kathmandu"
            });
            console.log(person);
            console.table(person);  // it will print the object in table format

Accessing Object properties :

There are two ways to access the properties of an object :

  1. Dot notation
  2. Bracket notation

1. Dot notation : It is the most common way to access the properties of an object.
syntax : object_name.property_name;

            Example :
            
                let user = {
                    name : "Shyam",
                    age : 25,
                    area : "Lalitpur"
                }
                console.log(user.name);
                console.log(user.age);
                console.log(user.area);

2. Bracket notation : It is used to access the properties of an object when the property name is not a valid identifier or when the property name is stored in a variable.
syntax : object_name["property_name"];

            Example :
            
                let student = {
                    sname : "Abhi",
                    sage :19,
                    susn : "4EG22AI001",
                    "srank": 2
                }
                console.log(student["sname"]);
                console.log(student["sage"]);
                console.log(student["susn"]);
                console.log(student["srank"]);

Updating, adding, deleting Properties :

adding syntax : object_name.adding_property = value
modifying syntax : object_name.exiting_object_property = value
deleting syntax : delete exiting_object_property 
            Example : 
            
                let car ={
                    brand : "KIA",
                    model : "XUV",
                    mileage : "18 per liter",
                    price : "15 lakhs",
                    headlight : "2"
                }
                console.log(car);
                console.table(car);

                //updating or modifying value
                car.model = "Compact SUV";
                console.log(car.model);
                console.table(car);

                //adding new property
                car.color = "white";
                console.log(car);
                console.table(car);

                //deleting one property
                delete car.headlight;
                console.log(car);
                console.table(car);
